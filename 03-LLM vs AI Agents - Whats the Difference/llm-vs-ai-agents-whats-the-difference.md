# LLM vs AI Agents - Whats the Difference?

#### AI vs Agent

**AI (LLM)** -

- jo bhi chezze hai voh brain karta hai
- kya ho agar brain ko body se alag kar do..
- LLM is --> brain kuch bhi input de do voh uska output de deta hai

> That LLM brain doesn't have body.
> E.g. - GO twitter and post a tweet 2+2

- brain ko pata hai kaise karna hai
- becz, its doesn't have body it can't perform anything

- Then agent come into picture.

**Agent** -

- Agent mains smart brain ko ek body do (make extra task also)
E.g. twitter ke upar jake kkuch post karna, files ko change karna, realtime weather fetch karna

#### Tools (LLM + agentTool)
- As a developer we make tools
E.g. - I can create tool as post a tweet, get weather info
- Agar user ne bola current weather ko fetch karo aur usko twitter ke upar ja ke post kar do
- This LLM 1st call `getWeatherInfo()` func as developer we can create this.
- Then use `postATweet()` this function actually do that tweet.

> Actually Agent is ready to perform specific task using their brain & body(your body)
